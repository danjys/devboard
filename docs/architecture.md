## Overview

DevBoard is an **Electron + React** desktop application that demonstrates:

- React as a UI framework for desktop apps via Electron
- CSS Modules for component-scoped styling with a shared design-token system
- React Context + useReducer for predictable state management (no Redux)
- HTML5 Drag and Drop API for inter-column task movement
- A contextBridge-based IPC pattern for secure Electron ↔ React communication
- Auto-generated architecture docs and Mermaid diagrams on every commit

## State Management

**TaskContext** (`src/renderer/context/TaskContext.jsx`) is the single source of truth for all task data.

- Uses `React.createContext` + `useReducer` — no external state library
- **Actions**: `ADD_TASK`, `UPDATE_TASK`, `MOVE_TASK`, `DELETE_TASK`, `REORDER_TASKS`
- **Persistence**: tasks are serialised to `localStorage` on every state change and hydrated on first render
- Consumed via the `useTasks()` custom hook

## Styling Conventions

| Convention | Details |
|------------|---------|
| **CSS Modules** | Every component has a co-located `.module.css` file — zero class-name collisions |
| **Design tokens** | All colours, spacing, typography defined as CSS custom properties in `globals.css` |
| **Theming** | Light/dark themes switch by setting `data-theme` on `<html>` — no JS re-renders needed |
| **No CSS-in-JS** | Keeps bundle lean and styles inspectable in DevTools |
